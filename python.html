Análise Vacinação COVID-19 pelo Mundo

Análise das Queimadas da Amazônia


Projeto dedicado para comparar o número de focos de calor entre a região norte do Brasil em relação a quantidade total do país.

O estudo foi feito através de uma base de dados em Excel disponibilizada pelo IBGE.

Nesse projeto, utilizei Python para importar os dados, obter a correlação entre eles e criar a figura.

Optei por calcular a correlação entre os dados para identificar se existe uma relação direta com o aumento no número de focos de calor na região norte em relação ao restante do país. O resultado foi de mais de 90% de relação direta.
